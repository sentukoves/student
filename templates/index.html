Для того, чтобы выполнить SELECT по БД ВСЕХ элементов необходимо ставить после адреса /all
Для того, чтобы выполнить поиск по табельному номеру, необходимоставить после адресса /tabnum?tabnum=Номер
Для того, чтобы выполнить UPDATE записи по табельному номеру, /update?balance=количество?tabnum=табельный номер кому перевести?fromtabnum=кто переводит
Получить json авторизации /auth?login=ADMIN&password=ADMIN примерный JSON ответа {"Status": {"Boolean": "True", "Category": "admin"}} True означет доступ разрешен
