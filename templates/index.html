Для того, чтобы выполнить SELECT по БД ВСЕХ элементов необходимо ставить после адреса /all
Для того, чтобы выполнить поиск по табельному номеру, необходимоставить после адресса /tabnum?tabnum=искомый номер
Для того, чтобы выполнить UPDATE записи по табельному номеру, /update?balance=количество&tabnum=табельный номер
