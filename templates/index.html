Для того, чтобы выполнить SELECT по БД ВСЕХ элементов необходимо ставить после адреса /all
Для того, чтобы выполнить поиск по табельному номеру, необходимоставить после адресса /tabnum?tabnum=Номер
Для того, чтобы выполнить UPDATE записи по табельному номеру, /update?balance=количество&tabnum=табельный номер кому перевести&fromtabnum=кто переводит для админской учетки /update?balance=10&tabnum=5435345&admins=True
Получить json авторизации /auth?login=ADMIN&password=ADMIN Примерный JSON ответа {"Status": {"Boolean": "True", "Category": "admin"}} , True означает доступ разрешен.
Вывести историю по табельному номер: /history?tabnum=5435345
Вывести всю историю: /history
Вывести все призы: /priz?all=True
Купить товар: /priz?tabnum=Табельный номер кто покупает&id=Номер в базе данных продукта (приходит в json)&count=Количество сколько человек хочет купить
